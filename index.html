Summary: WeatherPy

What’s the weather like as we approach the equator?


To answer this question, the WeatherPy project was created. The purpose of this project was to analyze how weather patterns may change and alter as one travels closer to the equator.

WheatherPy was executed using a Jupyter Notebook, the Python library, and the OpenWeatherMap API to create a representative model of weather across world-wide cities. A Python script was constructed to visualize the weather of 500+ cities across the world of varying distances from the equator.


The Objective:

Build a series of scatter plots, using Matplbotlib, to showcase the following relationships:

  • Latitude vs. Cloudiness (%)
  • Latitude vs. Humidity (%)
  • Latitude vs. Max Temperature (F°)
  • Latitude vs. Wind Speed (mph)
